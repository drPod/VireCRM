# Stripe Tax

すべての取引の売上税、VAT、GST 法令遵守を自動化します。

Stripe Tax は、税金義務の監視、登録、計算、徴収、申告を処理します。Stripe で支払うプラットフォームを通じて、1099 納税申告書の対象になることがあります。[納税申告書の受け取り](https://support.stripe.com/express/topics/1099-tax-forms)方法の詳細をご確認ください。

## カスタム決済フロー用の Stripe Tax API

Stripe 以外で処理された支払いも含め、すべての支払いに対する税額を計算し、レポートを作成します。
[詳しくはこちら](https://docs.stripe.com/tax/payment-intent/custom.md)
## 仕組み

[Stripe Tax の仕組み](https://docs.stripe.com/tax/how-tax-works.md): Stripe Tax を利用して税金に関する法規制への遵守を自動化する方法をご紹介します。

[Stripe による税金の計算方法](https://docs.stripe.com/tax/calculating.md): Stripe Tax で取引の税金を計算する方法をご紹介します。

[対応国](https://docs.stripe.com/tax/supported-countries.md): Stripe Tax が対応している国を確認し、各地域で課税される税金の計算と回収についてご確認ください。

## 今すぐ始める

[Stripe Tax を設定する](https://docs.stripe.com/tax/set-up.md): 使用を開始するには、ダッシュボードでいくつかのステップを実行して税金設定を行います。

[納税義務を監視する](https://docs.stripe.com/tax/monitoring.md): 過去の取引に基づいて、税金を徴収するために登録する必要がある場所を特定できます。

[税務登録の管理](https://docs.stripe.com/tax/registering.md): 税金を徴収するために、決済処理を開始したらすぐに登録を行います。

[申告と納税](https://docs.stripe.com/tax/filing.md): Stripe Tax を使用した税務報告と納税申告についてご紹介します。

## Stripe Tax を有効にする

[Payment Links での税金徴収](https://docs.stripe.com/tax/payment-links.md): コードを記述することなく、決済ページで税金を計算して徴収する方法をご紹介します。

[構築済み決済ページでの税金徴収](https://docs.stripe.com/payments/checkout/taxes.md): Stripe Checkout を使用して、1 回限りの支払いと継続支払いに課税する税金を計算して徴収します。

[サブスクリプションでの税金徴収](https://docs.stripe.com/tax/subscriptions.md): 継続支払いの税金を徴収および申告する方法をご紹介します。

[請求書での税金徴収](https://docs.stripe.com/tax/invoicing.md): 請求書に課税される税金を自動的に計算して徴収する方法をご紹介します。

[PaymentIntents で Tax API を使用する](https://docs.stripe.com/tax/payment-intent/custom.md): Stripe 決済における税額計算、トランザクション、差戻しを完全に制御できます。

[Stripe 外の決済で Tax API を使用する](https://docs.stripe.com/tax/off-stripe.md): サードパーティーまたは自社システムで処理された決済の税額計算とレポートを行えます。

[Connect で Stripe Tax を使用する](https://docs.stripe.com/tax/connect.md): Connect を使用して、プラットフォーム又はマーケットプレイスとして Stripe Tax を導入します。

## 機能

|  |
|  |
| [エンドツーエンドの法令遵守ソリューション](https://docs.stripe.com/tax/how-tax-works.md) | 直接または導入したパートナーを通じて、税務計算、登録、申請を管理することができます。                                                                                                          |
| [自動税計算](https://docs.stripe.com/tax/calculating.md)                  | [製品タイプ](https://docs.stripe.com/tax/tax-codes.md)、[顧客とビジネス場所](https://docs.stripe.com/tax/supported-countries.md)に基づいて税率を決定し、売上税、VAT、GST を瞬時に計算します。 |
| [複数国サポート](https://docs.stripe.com/tax/supported-countries.md)        | 国際的な消費税計算や越境取引に対応し、複数国の法令遵守をサポートします。                                                                                                                |
| [しきい値の監視](https://docs.stripe.com/tax/monitoring.md)                 | ローカルな税務登録の閾値に照らし合わせて売上を追跡し、税務上の義務が生じるとアラートを受け取ることができるため、ビジネスの成長に合わせてコンプライアンスを維持できます。                                                                |
| [税務登録管理](https://docs.stripe.com/tax/registering.md)                 | グローバルな税務登録の管理に役立ちます。Stripe を使用して、お客様に代わって[売上税を登録](https://docs.stripe.com/tax/use-stripe-to-register.md)することもできます。                                  |
| [商品税コード](https://docs.stripe.com/tax/tax-codes.md)                   | 商品やサービスを分類して正しく課税することで、異なる管轄区域でも正確な税額計算が可能になります。                                                                                                    |
| [プラットフォームとマーケットプレイスの組み込み](https://docs.stripe.com/tax/connect.md)    | Stripe Connect を使ってプラットフォームやマーケットプレイスと組み込まれ、連結アカウントに代わって税金計算や回収を行うことができます。                                                                          |
| [柔軟な料金体系オプション](https://stripe.com/tax/pricing)                       | 使用量ベースまたはサブスクリプションベースの料金体系モデルから、ビジネスニーズに合ったものを選択できます。使用状況と導入タイプに応じて透明性の高い料金が設定されます。                                                                 |
